# Run Manifest

> Each baseline / targeted real run is registered here with its metadata; `results-*.md` holds the scoring conclusions and hard metrics, this file holds "how it was run".
> Raw model outputs live locally in the maintainer's `tasks/current/eval-runs/` (gitignored) and are not committed by default; reproduce externally with the commands in `automation/eval/README.md`.

## Status

This is the German-English fork (`klartext`).
