# Deutsche Floskel-Liste / German Banned Phrases

> Quellen: Beobachtung alltäglicher KI-Ausgaben auf Deutsch, Nominalstil-/Behördendeutsch-Kritik, LinkedIn-/Ratgeber-KI-Deutsch, Community-Feedback (heise, r/de, ComputerBase), Übersetzungs-/Post-Editese-Forschung, Wikipedia:Anzeichen für KI-generierte Inhalte (CC-BY-SA, nur paraphrasiert, kein Textübernahme).

This table lists representative items only; it does not chase every synonym. For a new phrasing not listed, first check whether it is a variant of an existing pattern; only add a new entry when no pattern fits, or the false-positive boundary changes.

Diese Liste ist auf Deutsch, weil sie deutsche KI-Marotten katalogisiert. Die Anleitung drumherum ist bewusst Englisch (English-first meta language); die Zieltexte, die gereinigt werden, sind Deutsch und Englisch.

## Tier 1: Replace by default / Standardmäßig ersetzen

These appear far more often in AI text than in human text. Replace by default, but keep them under false-positive protection (see `severity.md`).

### Einleitungsfloskeln / Throat-clearing openers
- Es ist wichtig zu beachten, dass → löschen, direkt sagen
- Es ist wichtig zu betonen, dass → löschen
- Es sei darauf hingewiesen, dass → löschen
- Es lässt sich nicht leugnen, dass → löschen
- Es liegt auf der Hand, dass → löschen
- Bekanntlich → löschen, oder konkrete Quelle nennen
- In der heutigen schnelllebigen Welt → löschen oder konkreten Zeitbezug geben
- In der heutigen Zeit → löschen
- In einer Welt, in der → löschen
- Im Zeitalter von → löschen oder konkret werden
- Mit dem stetigen Fortschritt von … → löschen oder sagen, was fortgeschritten ist
- Lassen Sie uns eintauchen → löschen
- Tauchen wir ein (in das Thema) → löschen
- Werfen wir einen Blick auf → löschen oder direkt zeigen
- Kommen wir nun zu → löschen
- Grundsätzlich lässt sich sagen → löschen
- Nicht zuletzt → löschen
- Um es auf den Punkt zu bringen → löschen, dann den Punkt sagen

### Renderndes Betonen / Emphasis crutches
- Und das ist erst der Anfang. → löschen
- Lassen Sie das einen Moment sacken. → löschen
- Das muss man sich auf der Zunge zergehen lassen. → löschen
- Punkt. (als Betonungssatz) → löschen
- Ganz ehrlich: (als Auftakt) → löschen
- Machen wir uns nichts vor. → löschen
- Merke: / Merken Sie sich das. → löschen

### Business-/Denglisch-Jargon
- leveragen → nutzen
- (Synergien) heben → zusammenarbeiten, verbinden
- aufs nächste Level heben → verbessern
- Deep Dive → genauer ansehen
- (jemanden) abholen → erklären, mitnehmen
- Learnings → Erkenntnisse, Lehren
- zielführend → hilfreich, sinnvoll
- ganzheitlich → umfassend, vollständig
- Mehrwert generieren → nützen, helfen
- Mehrwert bieten → nützlich sein
- auf Augenhöhe → gleichberechtigt
- Stakeholder abholen → die Beteiligten einbinden
- Gamechanger → wichtige Änderung
- Low-Hanging Fruits → einfache erste Schritte
- committen (sich committen) → sich festlegen, zusagen
- aligned sein → einig sein, abgestimmt sein
- Empowerment / empowern → befähigen, ermöglichen
- disruptiv → grundlegend neu (nur wenn wirklich so)
- skalieren (als Floskel) → wachsen, ausweiten

Keep literal technical uses. `Der Dienst skaliert horizontal auf drei Knoten.` bleibt.

### Aufgeblähte Verben / Nominalstil (Streckformen auflösen)
- eine Optimierung vornehmen → optimieren
- eine Verbesserung erzielen → verbessern
- zur Anwendung bringen → anwenden
- zum Einsatz kommen → eingesetzt werden, benutzt werden
- in Angriff nehmen → anfangen, angehen
- Berücksichtigung finden → berücksichtigt werden
- eine Entscheidung treffen → entscheiden
- eine Überprüfung durchführen → prüfen
- unter Beweis stellen → zeigen, beweisen
- in Erwägung ziehen → erwägen, überlegen
- der Umsetzung dienen → umsetzen
- Anwendung finden → verwendet werden

### Bedeutungs-Inflation / Significance inflation
- von entscheidender Bedeutung → wichtig, oder sagen warum
- von zentraler Bedeutung → wichtig
- bahnbrechend → neu
- revolutionär → neu (Vergleich geben)
- wegweisend → wichtig
- richtungsweisend → wichtig
- ein echter Meilenstein → ein wichtiger Schritt
- nicht zu unterschätzen → wichtig, oder sagen wofür
- unverzichtbar (in der heutigen Zeit) → nützlich, oder sagen wofür
- ein echter Gamechanger → eine wichtige Änderung
- spielt eine entscheidende Rolle → hilft bei, ist Teil von
- unterstreicht → zeigt
- verdeutlicht (als Aufwertung) → zeigt
- zeugt von → zeigt
- ebnet den Weg für → ermöglicht
- hebt … auf ein neues Niveau → verbessert
- setzt neue Maßstäbe → ist besser bei (konkret)

### Kopula-Vermeidung / Copula avoidance (einfach "ist/hat")
- stellt ein … dar → ist ein
- fungiert als → ist
- dient als → ist
- bildet das Fundament für → ist die Basis von
- verkörpert → ist, steht für
- präsentiert sich als → ist
- weist … auf → hat
- verfügt über → hat

### Füllphrasen / Filler phrases
- im Rahmen von → bei, für
- im Hinblick auf → für, zu
- in Bezug auf → zu, bei
- vor dem Hintergrund (dass) → wegen, weil
- aufgrund der Tatsache, dass → weil
- zum jetzigen Zeitpunkt → jetzt
- zu einem späteren Zeitpunkt → später
- es versteht sich von selbst, dass → löschen
- an dieser Stelle sei erwähnt → löschen
- wie bereits erwähnt (als Füller) → löschen oder knapp rückverweisen

### Sycophantisch / meta
- Großartige Frage!
- Sehr gute Frage!
- Das ist eine ausgezeichnete Frage!
- Da hast du völlig recht!
- Da haben Sie absolut recht!
- Aber gerne!
- Selbstverständlich!
- Natürlich, gerne!
- Ich hoffe, das hilft dir weiter!
- Ich hoffe, das konnte deine Frage beantworten!

### Übersetzungs-Kalken / Calqued English idioms

English idioms translated word-for-word — grammatically fine, but idiomatic anomalies a native speaker rarely produces. A tell of translationese / post-editese (German drafted from an English prompt).

- außerhalb der Box denken → quer denken, neu denken
- am Ende des Tages → löschen, direkt den Punkt sagen
- das Beste aus beiden Welten → beide Vorteile zusammen
- den Ball ins Rollen bringen → anstoßen, starten
- die Extrameile gehen → sich besonders reinhängen
- der Elefant im Raum → das offensichtliche Problem
- auf derselben Seite sein → einig sein, abgestimmt sein

Guard: real German speakers do drop one of these now and then. Flag when they cluster, or sit in otherwise formal AI-German; keep a single one in genuinely casual speech, and keep any inside a direct quote.

## Tier 2: Flag when 2+ appear in one paragraph

Legitimate individually; clustering signals AI. Length reference: short paragraph (< 100 words) at 2+, long paragraph (≥ 100 words) at 3+.

- zudem, darüber hinaus, des Weiteren, ferner, überdies
- gleichzeitig, zugleich, folglich, somit, mithin
- maßgeblich, essenziell, elementar, grundlegend
- facettenreich, vielschichtig, vielfältig, mannigfaltig
- nahtlos, reibungslos, mühelos
- dynamisch, transformativ, wegweisend, zukunftsweisend
- Eckpfeiler, Grundpfeiler, Fundament (metaphorisch)
- immens, enorm, gewaltig (als Aufwertung)

## Tier 3: Flag at high density only

Common words, only a problem at high density. Thresholds: 3+ in short text (< 200 words), 5+ in medium (200–1000), > 0.5% in long text (> 1000). See `severity.md`.

- wichtig, entscheidend, zentral, wesentlich
- effektiv, effizient, leistungsstark, leistungsfähig
- innovativ, umfassend, ganzheitlich, robust
- nachhaltig, skalierbar, zukunftssicher, modern
- optimieren, verbessern, ermöglichen, gewährleisten, sicherstellen

## Füllwörter / Modal- und Abtönungspartikeln

Most of these are filler. Delete or rephrase — but German particles (`ja`, `doch`, `mal`, `halt`) are often natural in `chat`; only cut when they stack or inflate.

- natürlich, selbstverständlich, freilich
- im Grunde (genommen), im Endeffekt, letztendlich, letztlich
- quasi, sozusagen, gewissermaßen, praktisch
- tatsächlich, buchstäblich, regelrecht, wortwörtlich
- durchaus, zweifellos, ohne Frage, mit Sicherheit
- bekanntlich, offensichtlich, klarerweise
- interessanterweise, bemerkenswerterweise, erstaunlicherweise

## Denglisch-Hinweis / Code-switching note

For English words inside German sentences, judge by the meaning in that sentence, not by the English table:

- `Der eigentliche Leverage-Punkt liegt im Cache.` → `Leverage` here is business jargon, handle it.
- `Wir shorten mit 10x Leverage.` → `Leverage` here is a finance term, keep it.
- `context` in `der Context reißt nicht ab`, `p99` in `p99-Spitze` are technical terms — keep them, don't treat the mixed English as register mixing.
